# huntbox

## Setup
````
1. run ./install.sh

2. run virt-manager 
2.1 add new image with "+" icon on GUI,
2.2 select 'Local install media' and click next,
2.3 find your Kali ISO and untick the 'automatically detect OS' option and choose Debian 10 
2.4 adjust memory, disk and cpu
2.5 add own network (optional)

3. install kali using the guided installer
````
